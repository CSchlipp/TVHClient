Open Source
TVHClient verwendet folgende externe Bibliotheken:
  • Library for In-App billing : IAB v3 Library
  • Dialogs with material design : Material Dialogs
  • Date and time picker dialogs : datetimepicker
  • Material design navigation drawer : Material Drawer
  • View binding with Butterknife : Butterknife
  • Memory leak detection library : Leakcanary
  • Application crash reports : ACRA
  • Extensible logging library : Timber
  • Dependency injector : Dagger 2
