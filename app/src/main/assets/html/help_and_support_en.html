Casting
To get casting working properly certain things need to be considered. Transcoding must be enabled on the tvheadend server. This is usually the case on server versions > 4. Especially vaapi hardware assisted transcoding works best on recent tvheadend versions 4.3 and above.

A casting profile must exist with the following configuration:

  • Container WEBM/built-in
  • Video codec libvpx VP8, preset hq: nvenc(h264 / h265)
  • Audio codec libvorbis

Another profile for maximum compatibility with casting, browsers, external players and the internal player contains the following configuration:

  • Container av-lib/MP4
  • Video codec h.264 high profile
  • Audio codec AAC LC profile
