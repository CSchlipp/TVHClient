Kontakt
TVHClient wird entwickelt von Robert Siebert.
Bei Fragen oder Anregungen können Sie mich gerne kontaktieren unter rsiebert80@gmail.com
Die Teilnahme am Betatest, weitere Informationen und Diskussionen sind in der Google+ Gruppe möglich.
Fehlermeldungen und neue Features werden gerne in Github angenommen.
Kauf der Vollversion
Sie können die Vollversion per Inapp-Kauf freischalten. Bei der Transaktion gelten die Nutzungsbedingungen von Google.
E-Mail
Wenn Sie per mich per E-Mail kontaktieren, werden Ihre aus der Mail bzw. Anfrageformular inklusive der angegebenen Kontaktdaten zwecks Bearbeitung gespeichert. Diese Daten werden nicht ohne Ihre Einwilligung weitergegeben. Sie können jederzeit Auskunft über ihre gespeicherten Daten erhalten oder die Löschung der Daten beantragen.
