Anforderungen
  • Du benötigst die Vollversion von TVHClient.
  • Dein Server muss Transcodierung unterstützen.
  • Transcodierung, insbesondere VAAPI hardwaregestützte Transcodierung, funktioniert am besten mit den neuesten instabilen TVH 4.3 Versionen.
Einrichtung der Transkodierung

Für maximale Kompatibilität (Casting, die meisten Browser, Player und der interne Player) verwenden Sie ein Stream-Profil mit den folgenden Einstellungen:

  • Container: av-lib/MP4 (Beispiel)
  • Video codec: H.264, High profile (Beispiel)
  • Audio codec: AAC LC profile (Beispiel)
TVHClient einrichten
  • Sie können entweder eine IP-Adresse oder einen DNS-Namen für die Verbindung verwenden. Vor dem Start des Castings wird der DNS-Name in eine IP-Adresse umgewandelt, so dass das Casting funktioniert.
  • Wähle das entsprechende Casting-Wiedergabeprofil aus.
